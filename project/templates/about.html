What is RoverSnap?



RoverSnap is a web app created with Flask, Python, and SQL that allows users to search for Mars Rover pictures and save it to their own profile. Users are also able to search for other user profiles to streamline interesting rover photos found by all users on the site. All photos are retrieved from the official NASA API.
This app was originally created for the final project of Harvard CS50x. I would like to thank their team for the help.
